# Weather App

It's my first React App fetching data from OpenWeatherMap API.

## Configuration

### .env

```shell
VITE_OPENWEATHER_API_KEY="" # https://openweathermap.org/api
```

## Dependencies

- Vite
- React
- Node.js 18

## Running application

### Locally

```shell
$ npm run dev
or
$ yarn dev
```
